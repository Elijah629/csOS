# csOS
csOS is a C# Operating System, written in [Mosa](https://github.com/nifanfa/MOSA-Core)

## Credits:

@Elijah629
@asterd-og

## Features:

- Graphics
- Graphical User Interface (GUI)
- Apps
- Apps Manager
- Theme Manager
- Mouse Manager
